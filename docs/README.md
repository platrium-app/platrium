# Platrium Docs
View the latest documentation at [https://docs.platrium.org](https://docs.platrium.org)

## Contributing to Docs
Just like the entire monorepo, the docs are also configured to use the Nx build system. To start the docs server locally, run `nx serve docs`. To build and validate without running, use `nx build docs`.

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Deployment
Platrium Docs are deployed on GitHub Pages.
