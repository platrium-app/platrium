Welcome to the official documentation for **Platrium**, the high-performance self-hosted file sync and sharing platform.

This documentation site is split into two main sections:

1. **User Guide**: Everything you need to know about setting up, deploying, and using Platrium for your personal or enterprise file sync needs.
2. **Developer Docs**: Deep dives into our architecture, API design, core engine, and cross-platform SDK for those who want to contribute or build integrations.

If you are a developer looking to understand how Platrium works under the hood, a great place to start is our [Build Pipeline](/docs/architecture/build-pipeline) overview!
